property weapon_cooldown¶

Returns the time until the unit can fire again, returns -1 for units that can’t attack. Usage: if unit.weapon_cooldown == 0:

self.do(unit.attack(target))

elif unit.weapon_cooldown < 0:

self.do(unit.move(closest_allied_unit_because_cant_attack))

else:

self.do(unit.move(retreatPosition))
